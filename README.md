# juanjoseesva.dev

Personal portfolio website built with Astro.

## About

Hey! I'm Juan, a full-stack developer with over 10 years of experience building web and mobile applications. I started coding when I was 15 and turned it into a career at 18. Since then, I've had the chance to work with startups and international companies alike, tackling everything from frontend interfaces to backend systems and cloud infrastructure.

## Tech Stack

- **Framework:** Astro
- **Styling:** CSS with CSS Variables
- **Fonts:** JetBrains Mono
- **Hosting:** AWS Amplify
- **Domains:** juanjoseesva.dev, juanjoseesva.com

## Features

- Dark/Light mode toggle with system preference detection
- Responsive design for mobile and desktop
- Blog section with dynamic routing
- Experience timeline

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```

## Deployment

Deployed automatically via AWS Amplify on push to `main` branch.
